# Data

| File | Description | Source |
| ---- | ----------- | ------ |
| `1976-2020-house.csv` | Historical U.S. House returns | [MIT Election Data and Science Lab](https://electionlab.mit.edu/data) |
| `1976-2020-president.csv` | Historical presidential returns | [MIT Election Data and Science Lab](https://electionlab.mit.edu/data) |
| `party_control.csv` | Historical seat counts, popular vote shares, and retirements by party | Compiled by hand. Retirement data from Wikipedia |
| `past_pres_approval.csv` | Presidential approval polling | American Presidency Project --- extracted from [spreadsheet](https://docs.google.com/spreadsheets/d/1iEl565M1mICTubTtoxXMdxzaHzAcPTnb3kpRndsrfyY/) |
